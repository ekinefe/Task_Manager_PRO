# TaskManagerPro – Ekin Efe Gungör

## Setup Instructions:
1. Install XAMPP or another local server with PHP & MySQL.
2. Create a database named `task_manager_pro`.
3. Import `task_manager_pro.sql` using phpMyAdmin.
4. Place all project files inside your web root (e.g., `htdocs/task_manager_pro`).
5. Open your browser and go to `http://localhost/task_manager_pro/public`.

## Features:
- User Registration & Login
- Task CRUD (Create, Read, Update, Delete)
- Categories per user
- Search by term + filters (priority, category, status)
- Reports: task stats, overdue tasks, category summary
- Task sorting
- Secure (PDO, password_hash, XSS safe)

## Usage:
- Register or login.
- Add categories before assigning tasks.
- Use the task dashboard to manage everything.
